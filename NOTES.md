# 开盘三K线趋势突破策略 — 知识笔记

本文记录项目开发过程中涉及的所有概念、讨论与决策逻辑。

---

## 策略类型

这套策略的准确名称是**日内动量突破策略（Intraday Momentum Breakout）**，不是严格意义上的趋势跟踪策略，但两者是近亲：

| | 趋势跟踪（Trend Following） | 日内动量突破（本策略） |
|---|---|---|
| 持仓时间 | 数天到数周 | 当天内（纯日内） |
| 入场依据 | 价格突破均线/通道 | 开盘前3根K线方向 |
| 盈利来源 | 骑住大趋势 | 开盘动量延续几小时 |
| 隔夜风险 | 有 | 无（收盘强平） |
| 共同点 | 都依赖方向性行情，震荡市都受损 | ← |

---

## 核心逻辑

每个交易时段（夜盘+日盘各一次）开盘后，观察前3根5分钟K线的收盘价：
- **连续递增（close1 < close2 < close3）** → 做多
- **连续递减（close1 > close2 > close3）** → 做空
- 方向不一致 → 不开仓

入场价 = 第3根收盘价（+1跳滑点），止损在第3根极值外1跳，止盈 = 止损距离 × 盈亏比。

---

## 盈亏比与胜率的关系

**核心公式**：期望值 = 胜率 × 平均盈利 − 负率 × 平均亏损

以当前参数（胜率43%，盈亏比1:2）为例：
> 期望 = 0.43×2 − 0.57×1 = **+0.29R**（每承担1份风险，平均赚0.29份）

这解释了为什么"胜率只有43%但还是赚钱"——靠盈亏比补偿低胜率。

盈亏比测试结论（LC+PS，日线N40）：

| RR比 | 胜率 | 收益 | 回撤 |
|------|------|------|------|
| 1:1 | 50% | ~0% | 6.3% | 手续费吃光利润 |
| 1:1.5 | 45% | 15% | 6.5% | |
| 1:2 | 43% | 24% | 8.0% | |
| **1:2.5（当前）** | 39% | **24%** | **5.0%** | 性价比最优 |
| 1:3 | 37% | 23% | 5.7% | 止盈偏远，强平多 |
| 1:4（移远目标） | 37% | 42% | 4.4% | 收益高但样本依赖，类波段逻辑 |
| 1:4（止损砍半） | 24% | 15% | 9.2% | 止损被噪音反复触发，更差 |

**结论：1:2.5 是最优盈亏比**。1:4移远目标收益高但置信度低，且与其这样不如持仓更久（日内策略边界模糊）；1:4止损砍半反而更差，开盘正常波动就被扫出局。
**技术注意**：止盈距离需从实际入场价（含滑点）算起，不是从bar3.close算，才是真实RR比。

## 动态仓位管理（每笔1%风险规则）

**核心思路**：不是"每笔开几手"，而是"每笔最多亏账户净值的1%"。

计算公式：`手数 = floor(账户净值 × 1% ÷ 单手止损金额)`

效果：
- 止损**窄**（如PS，单手亏600元）→ 1000÷600=1.6 → 开1手
- 止损**宽**（如AG，单手亏1300元）→ 1000÷1300<1 → 跳过
- 账户亏损后，下注金额自动缩小（保护剩余资金）
- 账户盈利后，下注金额自动放大（复利效应）

同时有**保证金上限**：所有持仓合计占用保证金 ≤ 账户净值50%，防止过度杠杆。

---

## Regime 依赖性（最重要的局限）

**Regime = 市场状态**，分两种：
- **趋势市（Trending）**：价格朝一个方向持续走，开盘动量能延续
- **震荡市（Choppy/Range）**：价格来回磨，开盘假突破后立刻反转

本策略在趋势市赚钱，在震荡市亏钱。这不是参数问题，是策略类型决定的根本特性。

**4月回撤案例**：
- 2026年1月-4月初：趋势市，策略10万→14万
- 2026年4月中-5月：转震荡，周胜率从57%跌到21%，连续4周亏损
- LC趋势向上、PS趋势向下，两品种同时假突破，亏损叠加

---

## 趋势过滤（解决Regime依赖的第一步）

**日线N日均线方向过滤**：
- 前一日收盘 > N日均线 → 涨势 → 只允许做多
- 前一日收盘 < N日均线 → 跌势 → 只允许做空

**防未来函数**：用`shift(1)`，信号触发时只用前一天及更早的均线值，不泄露当天信息。

稳健性检验结果（N=10/20/40均有效，回撤稳定降到7-10%）说明这个规律不是过拟合，是真实有效的过滤。日内均线（5分钟200根）无效——快均线在震荡期自己也反复翻，反而误伤好信号。

**最优参数**：日线N40，将信号过滤掉304笔（约73%），保留145笔真正顺势的交易。

## 品种选择原则

**关键教训：扩展品种前必须先验证固定1手有正期望。**

尝试扩展RB（螺纹钢）、M（豆粕）、MA（甲醇）、I（铁矿石），固定1手回测全亏损（胜率28-38%）。动态仓位反而因为止损金额小、开仓手数多（最高20手）将亏损放大数倍，导致账户-51.8%。

**当前品种选择（LC+PS）逻辑**：
- LC（碳酸锂）：新能源赛道，趋势性强，单手止损适中
- PS（多晶硅）：工业品，运动规律不同于LC，一定程度分散
- AG（沪银）：样本仅4个月（有夜盘导致数据根数快速消耗），单独N40收益+1.3%边际弱，暂停

**LC/PS信号独立性**：两个品种的信号完全独立生成，不互相影响。同时亏损是因为同时受同一宏观环境影响（震荡市），不是互相干扰。

## 与买入持有的对比

同期数据（2025-07 ~ 2026-05）：

| 方式 | 收益 | 风险控制 |
|------|------|---------|
| 策略（LC+PS，v3） | +23.7% | 最大回撤7.2%，日内不留隔夜 |
| 买入持有LC | +154.6% | 无控制，需承担全部下行 |
| 买入持有PS | -32.7% | 无控制 |

策略价值在于：PS跌33%的环境下仍盈利23.7%，体现了"做空能力"和"风险控制"的价值。LC单边牛市时买持更优，但策略不需要判断大方向，规则驱动。

## 数据局限

- **tqsdk免费账户上限8964根**：AG约4.3个月，LC/PS约10个月
- **LC上市2022年，PS上市2023年**：历史本就不长，任何数据源都无法提供更早数据
- **主连跨月拼接**（KQ.m@）：换月时有价格跳空，轻微失真
- **样本偏短的影响**：所有统计结论（胜率、盈亏比、最优参数）置信度有限，需更长样本验证

---

## 实盘模拟方向

策略天然适合定时执行：每个交易时段只需运行一次，15分钟等前3根K线收盘判断信号，有信号则持仓直到止盈/止损/收盘强平。

技术方案：tqsdk TqSim（免费模拟账户），cron定时在09:00（日盘）和21:00（AG夜盘）启动脚本，策略逻辑直接复用现有代码，只需将数据源从历史CSV换成实时K线推送。
